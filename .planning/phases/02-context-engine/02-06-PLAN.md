    **Repo path resolution:**

    Client profiles in `/opt/agency-workspace/client-profiles/*.md` contain a `Repo(s)` field in the markdown body (not frontmatter). It looks like:
    ```
    | Repo(s) | meeting-copilot, client-portal |
    ```
    or
    ```
    **Repo(s):** meeting-copilot
    ```

    Known workspace repos live under `/opt/agency-workspace/`. Map repo names to paths:
    - repo name "meeting-copilot" -> `/opt/agency-workspace/meeting-copilot`
    - repo name "client-portal" -> `/opt/agency-workspace/client-portal`
    - etc.

    ```python
    WORKSPACE_DIR = Path("/opt/agency-workspace")

    def resolve_repo_path(repo_name: str) -> Path | None:
        """Resolve a repo name to its workspace path. Returns None if not found."""
        candidate = WORKSPACE_DIR / repo_name.strip()
        if candidate.is_dir() and (candidate / ".git").is_dir():
            return candidate
        return None
    ```

    **Function `_find_repos_for_client(slug: str) -> list[Path]`:**
    - Read the client profile at `/opt/agency-workspace/client-profiles/{slug}.md`
    - Parse the markdown content for Repo(s) field using regex:
      - Table format: `\|\s*Repo\(s\)\s*\|\s*(.+?)\s*\|`
      - Bold format: `\*\*Repo\(s\):\*\*\s*(.+)`
    - Split matched value by comma, strip whitespace
    - Resolve each repo name via `resolve_repo_path()`
    - Return list of valid repo Paths

    **Function `fetch_git_activity(slug: str | None = None, repo_paths: list[Path] | None = None, days: int = 7) -> list[GitCommit]`:**

    1. If slug provided and repo_paths is None, resolve repos from client profile via `_find_repos_for_client(slug)`.
    2. If no repos found, return [].
    3. For each repo path, run:
       ```python
       proc = await asyncio.create_subprocess_exec(
           "git", "-C", str(repo_path), "log",
           f"--since={days} days ago",
           "--format=%H|%an|%aI|%s",
           "--no-merges",
           "-n", "20",
           stdout=asyncio.subprocess.PIPE,
           stderr=asyncio.subprocess.PIPE,
       )
       stdout, stderr = await proc.communicate()
       ```
    4. Parse each line: split by `|` into sha, author, date_iso, message.
    5. Build GitCommit objects with `repo_name = repo_path.name`.
    6. Combine commits from all repos, sort by date descending.
    7. Return top 20 commits.
    8. On any error (subprocess fails, parse error), log warning and return [].
    9. Timeout: 10 seconds per git command.
    10. Log at INFO: "Found {n} commits across {m} repos for slug '{slug}'"

    **Add `if __name__ == "__main__"` block:**
    - Takes slug from sys.argv[1] or defaults to "hafnia-financial"
    - Calls `asyncio.run(fetch_git_activity(slug))`
    - Prints results as JSON

    Use `logging.getLogger(__name__)`.
