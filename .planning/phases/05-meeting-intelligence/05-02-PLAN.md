Surface "Last time you discussed..." prior meeting context from existing Fireflies transcript data.

Purpose: When a meeting starts, the copilot should immediately surface what was discussed in previous meetings with the same attendees — topics, open action items, and decisions. This data already exists in the Fireflies TranscriptSummary objects loaded by the context engine (Phase 2). This plan extracts and structures it for display and LLM consumption.

Output: PriorMeetingContext model and extract_prior_context function, integrated into UnifiedMeetingContext and classifier prompt.
</objective>

  <name>Task 1: Create prior context extractor</name>
  <files>engine/intelligence/prior_context.py</files>
  <action>
    Create `engine/intelligence/prior_context.py` with:

    1. `PriorMeetingContext` Pydantic model:
       - `last_meeting_date: datetime | None` — when the most recent prior meeting occurred
       - `last_meeting_title: str | None` — title of the most recent meeting
       - `topics_discussed: list[str]` — extracted from summary.overview fields (deduplicated)
       - `open_action_items: list[str]` — action items from prior meetings (all, deduplicated)
       - `total_prior_meetings: int` — how many past meetings found
       - `to_prompt_text() -> str` method that generates readable text like:
         ```
         Last time you met with [attendee names] on [date]:
         Topics discussed: [topic1], [topic2]
         Open items from previous meetings:
         - [action item 1]
         - [action item 2]
         ```
         If no prior meetings, return "No prior meeting history found."

    2. `extract_prior_context(attendees: list[AttendeeContext]) -> PriorMeetingContext` function:
       - Collect all TranscriptSummary objects from all attendees' meeting_history
       - Deduplicate by transcript ID (same meeting may appear for multiple attendees)
       - Sort by date descending (most recent first)
       - Extract topics: use the summary (overview) field from each transcript, truncated to first sentence or 120 chars
       - Extract action items: collect all action_items from all transcripts, deduplicate by text similarity (exact match after lowercasing and stripping)
       - Set last_meeting_date and last_meeting_title from the most recent transcript
       - Return PriorMeetingContext

    Import AttendeeContext from context.models. Use standard logging.
    Add `if __name__ == "__main__"` test block that creates mock TranscriptSummary data and verifies extraction.
  </action>
  <verify>
    Run: `cd /opt/agency-workspace/meeting-copilot/engine && python -c "

from intelligence.prior_context import extract_prior_context, PriorMeetingContext

ts2 = TranscriptSummary(id='t2', title='Client Check-in', date=datetime(2026, 3, 10, tzinfo=timezone.utc), summary='Reviewed deliverables', action_items=['Send proposal'])

print('All assertions passed')
